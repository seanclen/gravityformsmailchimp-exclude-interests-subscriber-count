== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Do I need this plugin? =

This plugin might be helpful if you experience timeouts and issues with loading Intrest Categories (Groups) in the Gravity Forms Mailchimp Add-On Feeds.

= Will this affect the options loaded from Mailchimp =

No. This plugin optimizes requests to Mailchimp's API for better performance. All of the required data you need is still loaded.
